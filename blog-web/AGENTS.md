# 前端模块规则

## 前端规则

- 前端使用 Vue 3 + Vite + Element Plus
- 页面按 `portal` 和 `admin` 方向规划
- 接口调用统一放在 `api` 层
- 登录态统一管理，优先保持实现简单清晰
- 后台页面优先保证管理效率，不先追求复杂视觉效果
- 开发完成后优先做页面流程和接口联调验证
- 页面展示文案、按钮文案、提示信息、空状态说明默认使用中文
- 组件注释、页面说明、交互备注默认使用中文
- 前端规则文档标题、说明文字、交互说明默认使用中文
- 变量名、组件名、路由名等技术标识符保持常见英文命名习惯

## 前端目标

前端要服务于两个目标：

- 支撑博客正常使用
- 与后端真实联调，形成完整项目闭环
