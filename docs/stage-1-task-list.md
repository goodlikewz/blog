# 第一阶段任务清单

## 一、阶段目标

第一阶段目标不是做一个功能很多但空洞的博客，而是做出一个：

- 能运行
- 能发文章
- 能展示文章
- 能体现 Java 项目工程能力
- 能同步承载学习总结和面试内容的博客系统

## 二、阶段范围

### 2.1 必做功能

- 后台登录
- 文章管理
- 分类管理
- 标签管理
- 前台首页
- 文章列表
- 文章详情
- Markdown 渲染
- 搜索
- 学习路线入口
- 学习总结专区入口
- 面试复盘专区入口
- 项目实战专区入口

### 2.2 必做内容

- 至少 9 篇首批文章准备计划
- 学习总结、场景面试、项目实战至少各有内容入口

## 三、执行顺序

### 任务 1：工程初始化

- 初始化 `blog-server`
- 初始化 `blog-web`
- 生成 `sql/init.sql`
- 配置本地运行说明

完成标准：

- 后端可启动
- 前端可启动
- 数据库脚本可执行

### 任务 2：后台登录模块

- 管理员登录接口
- JWT 工具类
- Token 校验拦截
- 退出登录
- 前端登录页
- 路由守卫

完成标准：

- 登录成功后可进入后台
- 未登录不可访问后台页
- 退出后登录态失效

### 任务 3：分类管理模块

- 分类新增
- 分类编辑
- 分类删除
- 分类列表
- 分类排序字段维护

完成标准：

- 后台可维护分类
- 前台可读取分类

### 任务 4：标签管理模块

- 标签新增
- 标签编辑
- 标签删除
- 标签列表

完成标准：

- 后台可维护标签
- 前台可读取标签

### 任务 5：文章管理模块

- 文章新增
- 文章编辑
- 文章删除
- 草稿 / 发布 / 下线
- 分类绑定
- 标签绑定
- 文章类型
- 系列名称或专题标识

完成标准：

- 后台可管理文章全生命周期
- 可区分学习总结、面试复盘、项目实战等内容

### 任务 6：前台展示模块

- 首页
- 学习路线入口
- 最新文章
- 学习总结入口
- 面试复盘入口
- 项目实战入口
- 文章列表
- 文章详情

完成标准：

- 已发布文章可正常展示
- 首页体现学习型博客定位

### 任务 7：搜索模块

- 标题搜索
- 摘要搜索
- 分类搜索
- 标签搜索

完成标准：

- 前台搜索可用

### 任务 8：首批内容落地

- 学习总结文章
- 场景面试文章
- 项目实战文章
- 踩坑记录文章

完成标准：

- 第一版博客不是空壳
- 至少具备最低内容标准

- 完成项目基线文档
- 初始化 Git
- 初始化前后端工程骨架
- 生成 SQL 脚本

### 第 2 天

- 完成后台登录后端接口
- 完成 JWT 基础能力

### 第 3 天

- 完成前端登录页和登录联调

### 第 4 天

- 完成分类管理和标签管理

### 第 5 天

- 完成文章管理基础版

### 第 6 天

- 完成前台首页、文章列表、文章详情

### 第 7 天

- 补内容、补文档、做第一阶段周总结
