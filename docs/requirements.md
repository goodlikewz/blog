## 项目定位

这是一个以 Java 学习为主线、以个人博客为载体、以知识沉淀和面试展示为结果的学习型技术项目。

## 项目目标

- 做出一个可运行、可上线的个人技术博客
- 通过真实业务练习 Java 后端能力
- 沉淀学习内容，形成长期知识库
- 打造一个可用于面试展示的项目作品

## 第一阶段范围

- 后台登录
- 文章管理
- 分类管理
- 标签管理
- 前台文章列表
- 前台文章详情
- Markdown 渲染
- 搜索功能

## 第二阶段范围

- Redis 缓存
- 点赞功能
- 评论功能
- 图片上传
- 热门文章
- 操作日志

## 第三阶段范围

- RocketMQ 异步处理
- Spring Cloud 微服务拆分
- Gateway 网关
- Nacos 注册中心
- Elasticsearch 全文检索
- Docker 部署

## P0 闭环建议

建议第一版先完成：

- 后台登录
- 分类管理
- 标签管理
- 文章管理
- 前台文章列表
- 前台文章详情

原则：

- 先把核心链路跑通
- 先有可演示成果，再逐步增强
