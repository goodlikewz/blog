# Codex 提示词清单

## 1. 工程初始化

```text
按项目默认规范，初始化 blog-server 和 blog-web 工程骨架，并补齐基础目录结构、配置文件和数据库初始化脚本。
完成后请说明：
1. 创建了哪些核心文件
2. 如何启动前后端
3. 还缺什么环境条件
```

## 2. 后台登录模块

```text
按项目默认规范，实现后台登录模块。
范围只包括：登录、Token 校验、退出登录，不做权限细分。
请默认完成：
1. 后端接口和 JWT 相关代码
2. 前端登录页和登录态管理
3. 前后端联调
4. 本地自测
5. 当日日报更新建议
```

## 3. 分类管理模块

```text
按项目默认规范，实现分类管理模块。
要求支持：新增、编辑、删除、列表查询、排序字段维护。
完成后请说明验证方法。
```

## 4. 标签管理模块

```text
按项目默认规范，实现标签管理模块。
要求支持：新增、编辑、删除、列表查询。
完成后请说明接口和页面如何验证。
```

## 5. 文章管理模块

```text
按项目默认规范，实现文章管理模块。
要求支持：新增、编辑、删除、草稿、发布、下线、按状态筛选。
请同时补齐相关 SQL 变更和基础自测。
```

## 6. 前台展示模块

```text
按项目默认规范，实现前台文章列表和文章详情页。
要求支持 Markdown 渲染、代码高亮、分类标签展示。
请完成接口联调和页面验证。
```

## 7. 搜索模块

```text
按项目默认规范，实现文章搜索功能。
搜索范围包括：标题、摘要、分类名称、标签名称。
请说明当前方案和后续接入 Elasticsearch 的升级方向。
```

## 8. 代码审查

```text
review 当前实现，重点检查：
1. bug 风险
2. 行为回归风险
3. 扩展隐患
4. 缺失测试
请按严重程度列出问题。
```

## 9. 日报更新

```text
根据今天的改动，更新 docs/daily/今天日期.md。
内容包括：今日目标、实际完成、关键改动、自测结果、问题记录、明日计划。
```
